# Goal, Code, Verify, Reflect

This repository contains the Quarto source files for **Goal, Code, Verify, Reflect: Guided Activities for Learning R Workflow**.

The book is being built incrementally. The current workflow is:

1. Build a minimal Quarto book scaffold.
2. Establish the layout and shared styling.
3. Add one chapter or activity file at a time.
4. Render after each addition.
5. Fix problems before adding the next file.

The rendered book output is generated in `_book/` and is not intended to be edited directly.
